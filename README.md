# Img2Char

图片转字符串

## Introduction

非常简陋，输入图片地址，在程序运行目录下生成char.txt

## Feature

* 自豪地选用 bash, cmd, powershell

* 使用 Python 强力驱动

* 我的第一个 Python 项目
